# Odin Landing Page

Лендинг-страница из курса [The Odin Project — Foundations](https://www.theodinproject.com/lessons/foundations-landing-page).

## Структура страницы

1. **Header + Hero** — навигация и главный экран с заголовком, текстом и картинкой
2. **Information** — заголовок и 4 карточки в ряд
3. **Quote** — большая цитата с автором
4. **Call to Action** — призыв к действию с кнопкой
5. **Footer** — копирайт

## Стек

- HTML5
- CSS3 (Flexbox, CSS-переменные)

## Дизайн-палитра

| Цвет | HEX | Использование |
|------|-----|---------------|
| Тёмный фон | `#1F2937` | Header, Footer |
| Светлый текст | `#F9FAF8` | Заголовки на тёмном |
| Серый текст | `#E5E7EB` | Параграфы на тёмном, фон Quote |
| Акцентный синий | `#3882F6` | Кнопки, CTA |
| Тёмный текст | `#1F2937` | Тексты на светлых секциях |

## Как запустить

Открыть `index.html` в браузере.
